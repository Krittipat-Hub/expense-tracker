# Expense Tracker RESTful API (MongoDB)
A mini-project RESTful API for managing personal income and expenses.

## Tech Stack
- Node.js + Express
- MongoDB

## How to run
1. npm install
2. node index.js

## API Endpoints
- POST /register
- POST /expense
- GET /expenses/:user_id
- PUT /expense/:id
- DELETE /expense/:id
- GET /summary/:user_id
